Foreword

Welcome to jbmyo.xyz, my personal website. I coded it from scratch with pure HTML and CSS, without using any frameworks. The goal for this website is to showcase some side projects I made in class or in my free time. The source code for the website and all other personal projects are available on my github.

Music Visualizer

This project is a music visualizer idea I wanted to create. I used the p5.js library, which is an interpretation of Processing for Javascript. Read more about it on github. The default song is a beat created by my friend MadMaik. Try to click on the canvas for a surprise...
